# PCETS Dead Cat Matrix

Full documentation in `pcets/README.md` Section 10.

## Decision Matrix

| Signal | Dead Cat (avoid) | Real Overreaction (consider) |
|--------|-----------------|------------------------------|
| Cash runway | <6 months | >12 months |
| Prior run (T-30 to T-0) | >30% | <10% |
| Analyst response | Silent or downgrade | Defense within 24h |
| Institutional ownership | Selling (form 4/13F) | Adding or holding |
| Path forward | No milestone <12m | Clear next catalyst |
| Beta | >2.5 | <1.5 |
| SC11 Path Clarity | Score 0 | Score 2+ |

## Disqualification Rule
4 or more Dead Cat signals present → DISQUALIFIED regardless of headline catalyst quality.

## Most Dangerous Combination
Dead Cat + Bear Regime + Analyst Silence = near-certain loss. Never enter.
